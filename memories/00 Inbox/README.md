# Inbox

**Capture first, organize later.** Drop anything here — ideas, questions, half-formed thoughts, things to follow up on.

When processing this folder:
1. For each note: decide where it belongs (Project / Area / Resource / Archive)
2. Move it to the right folder
3. Link it from the relevant note
4. Delete drafts that are no longer useful

Use the `05 Templates/Inbox Capture.md` template for new captures.

---

*This folder should ideally be empty or near-empty. If it's full, it's time to process.*
